## Environment Variables

Create a `.env.local` file in the root directory with the following variables:

```env
# OpenAI API Key (required for GPT-4o Vision skin analysis)
OPENAI_API_KEY=sk-your-openai-api-key

# Supabase Configuration (if using Supabase)
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

### Getting an OpenAI API Key

1. Go to [OpenAI Platform](https://platform.openai.com/)
2. Sign up or log in
3. Navigate to API Keys section
4. Create a new secret key
5. Copy the key and add it to your `.env.local` file

## Features

### Live Skin Scanner (`/scan`)

The live skin scanner uses:
- **Real-time camera feed** with face detection
- **Auto-detection checks**: Face position, distance, lighting, and stability
- **GPT-4o Vision API** for comprehensive skin analysis
- **Automatic capture** when optimal conditions are met

The scanner automatically detects:
- Face position and centering
- Optimal distance (30-70% of frame)
- Lighting quality (brightness 80-200)
- Movement/stability
- Skin issues (acne, blackheads, pores, etc.)
