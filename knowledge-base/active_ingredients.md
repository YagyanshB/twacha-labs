# Active Ingredients for Acne Treatment

## Primary Active Ingredients:

### Salicylic Acid (BHA):
- **Concentration**: 0.5-2% (OTC), 2-5% (prescription)
- **Mechanism**: Exfoliates dead skin cells, unclogs pores
- **Best For**: Blackheads, whiteheads, mild acne
- **Frequency**: Daily to 2-3x/week depending on sensitivity

### Benzoyl Peroxide:
- **Concentration**: 2.5-10%
- **Mechanism**: Kills acne-causing bacteria, reduces inflammation
- **Best For**: Inflammatory acne, pustules
- **Frequency**: Start low (2.5%), increase gradually
- **Note**: Can bleach fabrics, may cause initial dryness

### Retinoids (Retinol/Tretinoin):
- **Concentration**: 0.025-0.1% (prescription tretinoin), 0.1-1% (retinol OTC)
- **Mechanism**: Increases cell turnover, prevents clogged pores
- **Best For**: Comedonal acne, prevention, anti-aging
- **Frequency**: Start 2-3x/week, build to nightly
- **Note**: Requires sunscreen, may cause initial purging

### Niacinamide:
- **Concentration**: 2-10%
- **Mechanism**: Reduces inflammation, regulates sebum production
- **Best For**: Oily skin, inflammatory acne, redness
- **Frequency**: Daily, morning and/or evening
- **Note**: Generally well-tolerated, can be combined with other actives

### Azelaic Acid:
- **Concentration**: 10-20%
- **Mechanism**: Anti-inflammatory, antimicrobial, reduces hyperpigmentation
- **Best For**: Inflammatory acne, post-inflammatory marks
- **Frequency**: Daily to twice daily
- **Note**: Good for sensitive skin, helps with redness

### Alpha Hydroxy Acids (AHAs):
- **Glycolic Acid**: 5-10% (exfoliation, brightening)
- **Lactic Acid**: 5-10% (gentler than glycolic, hydrating)
- **Best For**: Surface exfoliation, texture improvement
- **Frequency**: 2-3x/week

## Combination Guidelines:

### Safe Combinations:
- Niacinamide + Salicylic Acid
- Niacinamide + Retinoids
- Azelaic Acid + Most Actives

### Avoid Combining:
- Retinoids + High % AHA/BHA (use on alternate days)
- Benzoyl Peroxide + Retinoids (can be destabilizing)

## Ingredient Selection by Acne Type:

**Comedonal (Blackheads/Whiteheads)**:
- Salicylic Acid (primary)
- Retinoids (prevention)
- AHA (exfoliation)

**Inflammatory (Pustules)**:
- Benzoyl Peroxide (primary)
- Niacinamide (support)
- Azelaic Acid (alternative)

**Cystic/Nodular**:
- Requires prescription (Tretinoin, Oral antibiotics, Spironolactone)
- Topical treatments less effective
- Professional consultation recommended

## Application Protocol:
1. Start with one active ingredient
2. Introduce gradually (2-3x/week)
3. Monitor for irritation
4. Add second active after 4-6 weeks if tolerated
5. Always use sunscreen with actives
